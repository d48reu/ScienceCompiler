Senolytics / Aging Mini-Corpus

Question
Across aging-related preclinical contexts, do senolytic interventions consistently improve healthspan-related outcomes, or do some senolytics produce meaningful tradeoffs or tissue-specific harms?

Why this corpus
- It is a different scientific domain from butyrate/neuroinflammation.
- It tests whether the Science Compiler generalizes beyond gut-brain anti-inflammatory stories.
- It includes both positive and negative senolytic signals.

Current scope
- small abstract-level / summary-level corpus
- mostly mouse work
- includes D+Q, fisetin, navitoclax, and broader senolytic interventions
